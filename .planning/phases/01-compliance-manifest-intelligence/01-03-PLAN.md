# Plan 01-03: React Frontend — Upload, Dashboard, HITL

<objective>
Build the React + TypeScript frontend with Vite. Implement the complete user flow: BMR file upload with drag-and-drop → real-time extraction progress via SSE → HITL review of extracted ingredients → compliance traffic-light dashboard with remediation → packing list export. The frontend should be demo-ready for the 8-minute hackathon pitch.
</objective>

<must_haves>
- Vite + React + TypeScript project with modern tooling
- Drag-and-drop file upload with progress indicator
- SSE-powered real-time extraction status (step-by-step progress)
- HITL review screen: ingredient table with confidence indicators, confirm/reject
- Compliance dashboard: shipment-level rollup (red/amber/green) + expandable per-check breakdown
- Remediation guidance displayed for failing checks
- Packing list preview + export (only after HITL confirmation)
- Clean, professional UI suitable for hackathon demo
</must_haves>

**frontend/src/api/client.ts:**
```typescript
import axios from 'axios';

const api = axios.create({ baseURL: '/api' });

export const uploadBMR = async (file: File): Promise<UploadResponse> => {
  const formData = new FormData();
  formData.append('file', file);
  const { data } = await api.post('/upload', formData);
  return data;
};

export const getShipment = (id: string) => api.get<Shipment>(`/shipments/${id}`).then(r => r.data);
export const confirmExtraction = (id: string) => api.put<Shipment>(`/shipments/${id}/confirm`).then(r => r.data);
export const getCompliance = (id: string) => api.get<ComplianceReport>(`/shipments/${id}/compliance`).then(r => r.data);
export const exportPackingList = (id: string) => api.post(`/shipments/${id}/export`).then(r => r.data);
```

**frontend/src/api/sse.ts:**
```typescript
export function useJobStream(jobId: string | null) {
  const [status, setStatus] = useState<JobStatus>({ step: 'idle', progress: 0, message: '' });

  useEffect(() => {
    if (!jobId) return;
    const source = new EventSource(`/api/jobs/${jobId}/stream`);
    source.onmessage = (event) => {
      const data: JobStatus = JSON.parse(event.data);
      setStatus(data);
      if (data.step === 'complete' || data.step === 'error') {
        source.close();
      }
    };
    source.onerror = () => { source.close(); };
    return () => source.close();
  }, [jobId]);

  return status;
}
```
